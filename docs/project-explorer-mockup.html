③ 動作仕様メモ
・プロジェクト行をクリック → そのプロジェクトを開いてエクスプローラーを閉じる(編集中のものは「編集中」バッジ表示)
・フォルダ行の ▼/▶ クリック → 折りたたみ切替(状態は localStorage に保存され再起動後も復元)
・ドラッグ&ドロップ: 行をフォルダ行へ重ねるとハイライト表示 → ドロップで移動。折りたたみ中のフォルダに0.6秒ホバーで自動展開
・⋮ メニュー(プロジェクト): 開く / 名前を変更 / フォルダへ移動… / 削除
・⋮ メニュー(フォルダ): 名前を変更 / サブフォルダを作成 / フォルダへ移動… / 削除
・フォルダ削除 → 中身の一覧を確認ダイアログで提示し、承認後に中のプロジェクトごと完全削除
・並び順: フォルダ優先 + 五十音順の自動ソート(手動並べ替えなし)
・右下のボタンでダークモード/メニュー/削除確認ダイアログの表示を切り替えてデザインを確認できます
